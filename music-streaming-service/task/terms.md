# Техническое задание

## Цель проекта
Создание MVP музыкального стримингового сервиса с возможностью покупки подписки.

## Функциональные требования
### Пользовательская часть:
1. Регистрация/авторизация
2. Поиск музыки по названию/исполнителю
3. Прослушивание треков
4. Создание плейлистов
5. Система подписок и оплаты

### Административная часть:
1. Управление музыкальным каталогом
2. Просмотр статистики
3. Управление пользователями

## Технический стек
Frontend: React + TypeScript
Backend: Node.js + Express
База данных: PostgreSQL
Хостинг: Vercel / Railway
